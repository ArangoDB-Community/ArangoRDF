ArangoRDF
-------------

Welcome to the documentation for **ArangoRDF**, a Python adapter for ArangoDB_ & RDF_.

Requirements
=============

- ArangoDB version 3.9+
- Python version 3.8+

Installation
============

.. code-block:: bash

    ~$ pip install arango-rdf --upgrade
