ArangoRDF
---------

Convert RDF Graphs to ArangoDB, and vice-versa.


Requirements
============

- Python 3.9+
- ArangoDB 3.10+

Installation
============

Latest Release

.. code-block::

   pip install arango-rdf

Current State

.. code-block::

   pip install git+https://github.com/ArangoDB-Community/ArangoRDF
